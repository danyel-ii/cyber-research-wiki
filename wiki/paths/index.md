# Curriculum Paths

## Overview
This section is the learner-facing entry point for the wiki. Use these paths when you want to study a subject in order without bouncing through repetitive meta pages.

## How to use this section
- Start with one subject path.
- Read the pages in order.
- Follow optional branches only when you want more depth.
- Use source pages to check evidence after the main topic pages make sense.
