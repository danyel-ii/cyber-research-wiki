# Foundational Certifications Path

## Overview
This path is for broad, early-stage offensive-security learning that mixes terminology, common assessment concepts, and lighter practical framing. It is the best current alignment for certifications such as CEH and CompTIA PenTest+.

## Audience
- readers building a first structured offensive-security foundation
- learners comparing introductory certifications
- maintainers deciding where broad exam-aligned material belongs
