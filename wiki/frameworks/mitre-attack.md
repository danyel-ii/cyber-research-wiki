# MITRE ATT&CK

## Overview
MITRE ATT&CK is a behavior-oriented framework for modeling adversary tactics and techniques based on real-world observations.

## Why this page exists
This framework is likely to become the main normalization layer for:
- attack mappings
- exercise design notes
- detection and coverage analysis
- cross-source comparisons
