# OWASP WSTG

OWASP WSTG is the framework anchor for the web-testing side of the wiki. Use it for articles that explain, extend, compare, or apply the OWASP Web Security Testing Guide as a structured approach to web application testing.

## What belongs here
- articles about how WSTG organizes web testing
- framework comparisons and mappings
- articles that explain how to use WSTG in practice
- methodology pieces that are clearly rooted in the framework

## How this category is positioned
This is the one framework category in the eight-category backbone. It supports the web-testing branch without replacing the broader pentest model.
