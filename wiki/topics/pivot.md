# Pivot

Pivot covers how access moves. It includes articles on relaying, tunneling, remote execution, lateral movement paths, segmented-network traversal, and the practical infrastructure that carries an assessment from one boundary to the next.

## How to use this category
Use Pivot when the primary question is not whether access exists, but how that access can be extended, relayed, or carried across boundaries safely and intelligibly.

## What belongs here
- remote execution and lateral movement tooling
- relay and tunnel patterns
- segmentation traversal and access extension
- articles about movement paths between systems or network segments
